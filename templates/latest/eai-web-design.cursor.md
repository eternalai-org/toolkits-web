## Design Principles to Follow

1. **Visual Hierarchy** - Guide user's eye with size, color, contrast
2. **White Space** - Don't crowd elements, let design breathe
3. **Typography** - Clear hierarchy, readable fonts, proper line height
4. **Color Theory** - Complementary colors, proper contrast ratios
5. **Consistency** - Spacing system, color palette, component patterns
6. **Accessibility** - WCAG AA contrast, readable text sizes
7. **Mobile First** - Start with mobile, scale up to desktop
